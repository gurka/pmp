void TcpBackend::ConnectionAsio::close ( )
overridevirtual

Closes the connection.

Any ongoing async tasks will be aborted and the OnDisconnected callback will be called either in this context or in a later context.

The Connection instance may only be deleted when the OnDisconnected callback has been called.
