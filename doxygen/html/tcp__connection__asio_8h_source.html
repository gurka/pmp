32  // A message consists of a header and data
33  // The header is 2 bytes and the value is the data length
34  //
35  // Example: a message with 6 bytes of data has a
36  // header that is 0x06 0x00 (network byte order)
37  // the total message length is 2 + 6 = 8 bytes
38  //
39  // The longest message we can send has 2^16 - 1 bytes of
40  // data (the header would be 0xff 0xff) and a total length
41  // of 2^16 - 1 + 2 bytes.
42  //
43  // Make sure that these buffers are capable of handling
44  // maximum size messages. Note that the read buffer first
45  // reads the header and then the message data while the
46  // write buffer must be able to contain the whole message.
47  // This is the reason for the size difference.
